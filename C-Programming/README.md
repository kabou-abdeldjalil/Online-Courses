# C Programming – Online Course Practice

This folder contains the practical work completed during the following online course:

**Course:** Apprendre la programmation en C – Cours complet  
**Platform:** Udemy  
**Instructor:** Arnaud Mercier  
**Course Link:** https://www.udemy.com/course/apprendre-la-programmation-en-c-cours-complet/

The code files follow the structure and progression of the course and were written as part of the training exercises.

## Topics Covered

- Basic syntax and program structure
- Variables and memory management
- Conditions and loops
- Functions
- Pointers
- Arrays and strings
- Preprocessor directives
- Modular programming
- Custom types (struct, typedef)
- Dynamic memory allocation
- File handling

## Purpose

The objective of this course was to understand the fundamentals of C programming and low-level concepts such as memory management and structured program design.
